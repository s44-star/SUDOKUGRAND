# Sudoku Grand
> Premium mythic Sudoku — two modes, one world.

🔗 **Play:** [sudoku.lexconv.com](https://sudoku.lexconv.com)

---

## Two Modes

### ◇ Advanced — The Solitary Descent
Solo Sudoku with real stakes. Fill the 9×9 grid with 3 lives — every mistake costs you. Chain correct placements to build combos and light up the yantra.

### ◆ Elite — Duel The Ancient
Race a mysterious ancient rival to complete the same grid. Place numbers 1–5 correctly to earn combat powers. First to finish wins.

---

## How to Play
- Fill every row, column, and 3×3 box with numbers 1–9
- No repeats in any row, column, or box
- **Advanced:** 3 mistakes = game over
- **Elite:** The rival solves alongside you in real time
